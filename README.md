# BFC Flensburg Website

Website des Business and Finance Club Flensburg e.V.

## Setup

```bash
npm install
npm run dev
```

Öffne [http://localhost:3000](http://localhost:3000)

## Deployment (Vercel)

1. Push zu GitHub
2. vercel.com → "Add New Project" → Repo auswählen → Deploy
3. Domain: Settings → Domains → `bfc-flensburg.de` hinzufügen

## Formular

Mitgliedsanträge gehen über Formspree an info@bfc-flensburg.de.
